# 300B-TubeAmp-SoundEffect-for-Windows
This Windows application recreates the exact sound effect of the 300B vacuum tubes for your computer audio. Through precise physical modeling  (via APX555B measurements of a Manley Neo-Classic 300B Preamp), it brings the magic of tube sound within reach - allowing every Windows user to enjoy 300B sonic character in an immersive digital environment.
NOTE: This program is under development and will be uploaded soon.
